## How to Use

1. **Unlock Phone**: Click on the lock screen to unlock the device
2. **Launch Apps**: Tap any app icon on the home screen
3. **Quick Settings**: Click the menu button (‚â°) in the top-left to open quick settings
4. **Navigate**: Use the bottom navigation bar to go home, back, or view recent apps
5. **Adjust Brightness**: Use the brightness slider in quick settings
6. **Close Apps**: Tap "Back" button in any app to return to home screen
